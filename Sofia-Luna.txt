Stretch Questions

    1. What is the difference between an inline element and a block element?
    inline elements does not cause a line break while block elements always start in a new line and they take the full width of a page.

    2. What happens when an element is positioned absolutely? 
    If positioned absolutely that element would be taken out of the document and no space is created for that element in the page layout. It is positioned relative to the closest positioned ancestor. 
    Its final position is determined by the values of top, right, bottom, and left.

    3. How do I make an element take up only the amount of space it needs but also have the ability to give it a width? 
    By creating a CSS reset: 
    * {
        margin: 0;
        padding: 0;
    }

    4. Name 3 elements that are diplay block by default, 2 elements that are display inline by default and 1 element that is display inline-block by default
    Block elements: <header></header>, <h1></h1> - <h6></h6>, <ul></ul>, <li></li>
    in-line elements: <a></a>, <span></span>, <img>
    inline-block: <button></button>, <input>

    5. In your own words, explain the box model. What is the fix for the box model? 
    Each element is wrap by box, some elements have default settings and the box model let's us style elements in the way we want.
    The box model is composed by four parts, content, padding, border and margin. The padding controls the space inside the box, the border surrounds the box and the margin is the space outside the box.
